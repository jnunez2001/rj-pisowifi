## PART 1: COMPANY OVERVIEW

### What is Zentry Systems?

A **software platform company** providing integrated business management tools for small-to-medium cafes, internet service providers, and esports centers across Southeast Asia and beyond.

We're not a café operator (like Starbucks). We're a **software company** (like Square or Toast) that sells tools TO café operators.

### Why Zentry Systems?

**Problem We Solve:**

```
Current Situation:
├─ Small café owners buy 3-4 separate systems
│  ├─ WiFi rental software
│  ├─ POS system
│  ├─ PC rental system (if available)
│  └─ No integration = data silos + manual work
│
├─ Each system:
│  ├─ Has different user logins
│  ├─ Different customer databases
│  ├─ Separate reporting
│  └─ Expensive and fragmented
│
└─ Result: Café owner is frustrated, confused, lost in complexity

Our Solution:
├─ One integrated platform
├─ One login, one dashboard
├─ Unified customer data (WiFi + PC + POS)
├─ All reports in one place
└─ Simple, affordable, unified
```

### Company Values

```
1. Simplicity
   └─ Software should be easy, not complex

2. Integration
   └─ Products work together seamlessly

3. Reliability
   └─ Works offline and online, always available

4. Community
   └─ Built BY café owners, FOR café owners

5. Scalability
   └─ Works for 1 location or 100+ locations
```

## PART 5: DEPLOYMENT OPTIONS

### Option A: Mini PC as Server + Kiosk (Simplest)

```
Hardware:
├─ Mini PC (₱30-50K)
│  ├─ Runs ZenCafe server (backend)
│  ├─ Displays kiosk UI (frontend)
│  └─ Has touchscreen output
├─ Touchscreen 7-10" (₱8-12K)
├─ Bill acceptor (₱8-15K)
├─ Coin slot (existing or ₱5-8K)
└─ Receipt printer (optional, ₱5-8K)

Cost: ₱51-85K
Ideal For: Single-location cafes
```

### Option B: Server on Existing PC (More Flexible)

```
Hardware:
├─ Existing server PC (already owned)
│  ├─ Runs ZenCafe server (service/background)
│  └─ Handles all thin clients
├─ Mini PC (₱30-50K) = kiosk only
│  ├─ Displays touchscreen UI
│  └─ Connects to server PC
└─ Payment hardware (₱25-35K)

Cost: ₱55-85K (reusing existing server saves money)
Ideal For: Diskless setups, multiple locations
```

### Option C: Cloud Server (Multi-Location Hub)

```
Hardware:
├─ Mini PC at each location (₱30-50K each)
│  ├─ Local database
│  ├─ Works offline
│  └─ Syncs to cloud when online
├─ Cloud server (AWS/Azure/self-hosted)
│  ├─ Aggregates all location data
│  ├─ Central admin dashboard
│  ├─ Multi-location reporting
│  └─ Optional (can skip if local only)
└─ Payment hardware (₱25-35K per location)

Cost per location: ₱55-85K + cloud ops cost
Ideal For: Multi-location operators, franchising
```
