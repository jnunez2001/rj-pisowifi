## PART 1: COMPANY OVERVIEW

### What is Zentry Systems?

A **software platform company** providing integrated business management tools for small-to-medium cafes, internet service providers, and esports centers across Southeast Asia and beyond.

We're not a café operator (like Starbucks). We're a **software company** (like Square or Toast) that sells tools TO café operators.

### Why Zentry Systems?

**Problem We Solve:**

```
Current Situation:
├─ Small café owners buy 3-4 separate systems
│  ├─ WiFi rental software (₱20K/year)
│  ├─ POS system (₱30K/year)
│  ├─ PC rental system (if available, ₱50K+)
│  └─ No integration = data silos + manual work
│
├─ Each system:
│  ├─ Has different user logins
│  ├─ Different customer databases
│  ├─ Separate reporting
│  └─ Expensive ₱100K+/year total
│
└─ Result: Café owner is frustrated, confused, broke

Our Solution:
├─ One integrated platform
├─ One login, one dashboard
├─ Unified customer data (WiFi + PC + POS)
├─ All reports in one place
└─ Affordable: ₱5-30K/year (not ₱100K+)
```

### Company Values

```
1. Simplicity
   └─ Software should be easy, not complex

2. Affordability
   └─ Small businesses deserve good tools at fair prices

3. Integration
   └─ Products work together seamlessly

4. Community
   └─ Built BY café owners, FOR café owners

   └─ Works for 1 location or 100+ locations
```
